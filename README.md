# 🦷 OdontoHub

O **OdontoHub** é um sistema de gestão odontológica desenvolvido com o objetivo de otimizar o fluxo de atendimento em consultórios. Este projeto é parte da minha trajetória como desenvolvedor em formação, focado em criar soluções práticas para organização de dados de pacientes e procedimentos clínicos.

## 🚀 Tecnologias Utilizadas

*   **Front-end**: Interface construída com HTML5, CSS3 e JavaScript para uma experiência de usuário intuitiva.
*   **Back-end**: Servidor robusto desenvolvido em Python utilizando o framework Flask.
*   **Banco de Dados**: Gerenciamento de dados realizado via SQLite, garantindo simplicidade e eficiência.
*   **Hospedagem**: O sistema está em nuvem e disponível através da plataforma Render.

## 📋 Funcionalidades Principais

*   **Cadastro de Pacientes**: Sistema completo para registrar e gerenciar informações dos pacientes.
*   **Agendamento**: Gestão de agenda para controle de consultas e atendimentos.
*   **Lançamento de Procedimentos**: Registro detalhado dos procedimentos realizados em cada paciente.
*   **Dashboard em Tempo Real**: Atualização automática da quantidade de pacientes agendados no dia e do total de pacientes cadastrados no sistema.

## 🛠️ Como rodar o projeto localmente

1.  Clone este repositório:
    ```bash
    git clone [https://github.com/devPedro17/sistema-odontologico.git](https://github.com/devPedro17/sistema-odontologico.git)
    ```
2.  Entre na pasta do projeto:
    ```bash
    cd sistema-odontologico
    ```
3.  Instale as dependências necessárias:
    ```bash
    pip install -r requirements.txt
    ```
4.  Execute o servidor:
    ```bash
    python app.py
    ```
5.  Acesse `http://127.0.0.1:5000` no seu navegador.
